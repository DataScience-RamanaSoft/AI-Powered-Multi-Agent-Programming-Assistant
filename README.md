# CrewAI Multi-Agent Code Generation System

An AI-powered multi-agent software development system that automates Python code generation, debugging, and validation using CrewAI and locally hosted LLMs.

## 📌 Project Overview

This project demonstrates how multiple AI agents can collaborate to automate the software development lifecycle. Instead of using a single AI model, the system assigns different responsibilities to specialized agents such as code generation, debugging, and validation.

The project is built using CrewAI with a locally hosted LLaMA 3.1 model through Ollama, ensuring privacy, faster execution, and zero API costs.

## 🚀 Features

- Multi-agent collaboration using CrewAI
- Automated Python code generation
- Intelligent debugging and error correction
- Final code validation and review
- Local LLM execution using Ollama (LLaMA 3.1)
- Modular and scalable architecture
- No dependency on paid cloud APIs

## 🛠️ Technologies Used

- Python
- CrewAI
- Ollama (LLaMA 3.1)
- LiteLLM

## 🧠 System Workflow

The system follows a structured multi-agent workflow:

1. **Developer Agent**
   - Generates Python code from user requirements

2. **QA Engineer Agent**
   - Detects bugs and applies corrections

3. **Reviewer Agent**
   - Validates logic and verifies final output quality

This layered architecture improves accuracy and reliability compared to single-step AI code generation.

## 📂 Project Architecture

User Input → Developer Agent → QA Agent → Reviewer Agent → Final Output
